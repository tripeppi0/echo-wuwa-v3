POLITIQUE DE CONFIDENTIALITÉ

Application : Echo Wuwa v3
Dernière mise à jour : 28 décembre 2024
Contact : tripeppi0@gmail.com

1. PRÉSENTATION

Cette politique de confidentialité décrit comment cette application Google Sheets et son script Apps Script utilisent et protègent vos données.

2. COLLECTE DE DONNÉES

Cette application ne collecte AUCUNE donnée personnelle.

  • Le script fonctionne uniquement dans votre copie personnelle de la feuille
  • Toutes les données restent dans votre Google Drive
  • Aucune information n'est transmise à des serveurs externes
  • Aucun suivi ou analytics n'est effectué

3. UTILISATION DES AUTORISATIONS GOOGLE

L'application demande les autorisations suivantes :

3.1 Accès aux Google Sheets

  • Pourquoi : Pour lire et modifier les données dans votre feuille de calcul
  • Utilisation : Le script traite uniquement les données de votre copie de la feuille
  • Portée : Limité à la feuille que vous utilisez

3.2 Exécution en votre absence

  • Pourquoi : Pour les traitements longs qui nécessitent plusieurs étapes
  • Utilisation : Le script peut créer des déclencheurs temporaires pour reprendre automatiquement
  • Portée : Limité aux opérations dans votre feuille

4. STOCKAGE DES DONNÉES

  • Toutes vos données restent dans votre Google Drive
  • Aucune copie n'est créée sur des serveurs externes
  • Vous gardez le contrôle total de vos fichiers
  • Vous pouvez supprimer votre copie à tout moment

5. PARTAGE DES DONNÉES

Aucune donnée n'est partagée.

  • Vos données ne sont pas transmises à l'auteur du script
  • Vos données ne sont pas partagées avec des tiers
  • Le script fonctionne localement dans Google Apps Script
  • Chaque utilisateur travaille sur sa propre copie isolée

6. SÉCURITÉ

  • Le script utilise l'infrastructure sécurisée de Google Apps Script
  • Les autorisations sont gérées par le système OAuth 2.0 de Google
  • Vous pouvez révoquer les autorisations à tout moment depuis votre compte Google
  • Le code source peut être consulté dans Extensions > Apps Script

7. VOS DROITS

Vous avez le droit de :

  • Consulter le code du script (Extensions > Apps Script)
  • Modifier ou supprimer votre copie de la feuille
  • Révoquer les autorisations (myaccount.google.com/permissions)
  • Supprimer les déclencheurs créés (Extensions > Apps Script > Déclencheurs)

8. MODIFICATIONS DE CETTE POLITIQUE

En cas de modification de cette politique, la date de mise à jour sera actualisée en haut de ce document.
